Who - visit - me shows the  friends who visit to author's blog recently.
== Description ==
*Who - visit - me  display gravatar, visit time, blog link, the source address (IP real address ) for all friends who have ever left a comment when they visit the next time.<br>
*Display recent visitor in your blog sidebar.<br>
*If you have questions, Please contact [admin#winysky.com] or leave a message in my blog(http://winysky.com).<br><br>
*who-visit-me(最近来访）不是一个访问统计插件，本插件功能显示最近访问博客的朋友，类似于SNS网站“最近来访”功能。凡是曾在博客留言的朋友下次来访均可以显示头像、到访时间、博客链接、来源地址(ip真实地址-邪恶的功能）<br>
*可在后台设置参数<br>
*提供技术支持及咨询，联系邮件请发【admin#winysky.com】或者在我博客【http://winysky.com】留言<br>

== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the 'Who-Visit-Me' directory into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for Who-Visit-Me.
4. Go to the Who-Visit-Me Options page (Settings > Who-Visit-Me Option).
5. Go to widgets page to add the widget to your sidebar.
	Also you can use function <code><?php wvm_echo(); ?></code> at the place you want to show.

下载插件，上传到插件目录，在后台管理中激活插件，到设置页面进行简单设置，然后在小工具页面向边栏添加边栏回复小工具即可。
当然，你也可以直接在需要显示最新评论的地方调用<code><?php wvm_echo(); ?><code>。<br>
